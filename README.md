# outlier_detecion
Evaluation of the effectiveness of various methods designed to detect outliers (anomalies). Based on the Outlier Detection DataSets (ODDS) repository. Five methods were compared:
- isolation forest
- dbscan (primarly a clustering alghorithm but has built-in outlier detection)
- deadwood
- one-class svm
- local outlier factor. <br>
The main topics considered were evaluation of performance and robustness to hyperparameter changes and changes in data.
